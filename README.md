# Nutritional, genotoxic, and human health traits of the traditional milpa agroecosystem

Milpa Agroecosystem Study

This repository contains the scripts, data processing workflows, and statistical analyses used to evaluate the nutritional, genotoxic, and human health traits of the traditional milpa agroecosystem.

All scripts were executed in RStudio 4.3.3 (R Core Team, 2025). Key dependencies include:
vegan (Oksanen et al., 2015)
FactoMineR (Lê et al., 2008)
factoextra (Kassambara and Mundt, 2020)
performance
corrplot & Hmisc
